Test task InCode Group

## Getting started

    $ git clone https://github.com/otyunin/InCode-test.git
    $ cd InCode-test
    $ npm install
    $ npm run start
Go to *http://localhost:3000/*


## Scripts
Start client and server:

    $ npm run start
Start only client:

    $ npm run client
Start only server:

    $ npm run server
